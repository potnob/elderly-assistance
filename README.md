# elderly-assistance
Everpendant is a powerful care-support tool designed to help caregivers stay informed, connected, and prepared—every single day. Whether you're caring for a parent, grandparent, or anyone who needs a little extra support, Everpendant provides peace of mind through smart reminders, wellness updates, and reliable communication tools.
Key Features
Customizable Reminders
Set and share important reminders like medication schedules, appointments, or wellness tasks for both caregiver and care recipient. Alerts ensure nothing is ever missed. Emergency Contacts
Store essential phone numbers and call instantly with just a tap. We are developing smart alerts to automatically notify contacts during emergencies—like a fall.
Daily Check-Ins & Wellness Logs
Care recipients can complete a simple daily check-in, giving caregivers instant updates and a recorded history of wellbeing—accessible anytime.
Real-Time Location Sharing
Both parties can view device location when needed, helping ensure safety and quick response in urgent situations.
Secure by Design
Everpendant is built using Flutter & FlutterFlow, powered by Dart, with information securely stored in Firebase—so your data stays protected.
